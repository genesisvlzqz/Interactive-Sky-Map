About this project

The purpose of this project is to establish a web-app that is constantly up to date with confirmed planets data from the NASA Exoplanet Archives using the API. We are plotting stellar systems Declination vs Right Ascension using D3

Click and Drag on the Map to move around. Scroll to zoom in or out. Hover over the data points to view Name and planets in the system. Click the data point to view the Exoplanet Archive page for a system.
